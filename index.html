Politique de confidentialité

  1. Collecte de données personnelles

    Le Parti politique BUREC collecte des données personnelles telles que les noms, adresses, numéros de téléphone, adresses e-mail, opinions politiques et données de comportement. Ces données sont collectées dans le but de constitution de la base de données du Parti politique BUREC pour les faits liés aux activités du parti.

  2. Partage de données avec des tiers

    Le Parti politique BUREC peut travailler avec des tiers pour collecter et traiter les données, par exemple pour mener des enquêtes d'opinion ou pour envoyer des messages de campagne. Ces tiers sont autorisés à utiliser les données personnelles uniquement dans le cadre de la mission qui leur a été confiée par le Parti politique BUREC.

  3. Sécurité des données personnelles

    Le Parti politique BUREC prend des mesures de sécurité pour protéger les données personnelles qu'il collecte, notamment contre les pertes, les abus et les accès non autorisés.

  4. Droits des citoyens

    Les citoyens ont le droit d'accéder aux données collectées à leur sujet, de les corriger ou de les supprimer. Le Parti politique BUREC respecte les droits des citoyens et met en place les moyens nécessaires pour leur permettre d'exercer ces droits.

  5. Conservation des données personnelles

    Le Parti politique BUREC ne conserve les données personnelles que le temps nécessaire pour atteindre les objectifs pour lesquels elles ont été collectées. Les données personnelles sont supprimées dès qu'elles ne sont plus nécessaires ou conformément à la législation en vigueur.

  6. Modifications de la politique de confidentialité

    Le Parti politique BUREC se réserve le droit de modifier cette politique de confidentialité à tout moment. Les modifications seront affichées sur le site Web du parti et seront effectives dès leur publication.
